## usage

- 서브 패키지 폴더

  - 모바일: ./packages/climbingapp
  - 웹: ./packages/climbingweb

- 명령어
  - 모바일 실행
  ```bash
    - aos: yarn workspace climbingapp run android
    - ios: yarn workspace climbingapp run ios
  ```
  - 웹 프로젝트 개발모드 실행
  ```bash
    - yarn workspace cimbingweb run dev
  ```
